KALI LINUX & CEH PROFESSIONAL MASTER CHEATSHEET

This cheatsheet is structured from foundational knowledge to elite professional techniques.

CORE LINUX COMMANDS (FOUNDATIONAL)

FILE SYSTEM NAVIGATION
pwd - print working directory
ls - list directory contents
ls -la - list all files with details
cd - change directory
cd .. - move up one directory
cd ~ - go to home directory

FILE OPERATIONS
cat - display file content
less - view file page by page
head/tail - show first/last lines of file
touch - create empty file
cp - copy files
cp -r - copy directories recursively
mv - move or rename files
rm - remove files
rm -rf - force remove directories (DANGEROUS)
mkdir - create directory
find - search for files
locate - find files using database

TEXT MANIPULATION
grep - search text patterns
awk - pattern scanning and processing
sed - stream editor for filtering/transforming
sort - sort lines of text
uniq - report or omit repeated lines
cut - remove sections from lines
wc - word count

PERMISSIONS & OWNERSHIP
chmod - change file permissions
chown - change file owner
chgrp - change file group
umask - set default permissions

PROCESS MANAGEMENT
ps - process status
ps aux - detailed process list
top - dynamic process view
htop - enhanced process viewer
kill - terminate process
killall - kill processes by name
pkill - kill by pattern
nice - set process priority
renice - change running process priority

ADVANCED NETWORKING

socat - multipurpose relay

masscan - mass IP port scanner

SECURITY-SPECIFIC COMMANDS

strace - trace system calls
