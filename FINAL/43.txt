Topic 43: Types of Malware

Understanding the different types of malware is like learning the different kinds of weapons in an arsenal. Each has a specific purpose, method of operation, and set of indicators. A professional security analyst or ethical hacker must be able to identify and understand these categories to properly defend against them or use them in a controlled test.

Here are the primary types of malware, explained in detail:

1.  **Virus:**
    A virus is a piece of code that attaches itself to a legitimate program or file and requires user action to execute and spread. It is not a standalone program. Think of it as a biological virus that needs a host cell.
    - **How it Works:** It infects a host file (e.g., an .exe or .doc file). When a user runs the infected program, the virus code is executed. It can then replicate by infecting other files on the system.
    - **Primary Trait:** Requires human action to spread. It is self-replicating but not self-propagating.
    - **Example:** The ILOVEYOU virus spread by email as a love letter attachment (`LOVE-LETTER-FOR-YOU.TXT.vbs`). When opened, it infected the system and mailed itself to everyone in the user's address book.

2.  **Worm:**
    A worm is a standalone, self-replicating program that spreads across networks without any user interaction. It exploits vulnerabilities in operating systems or applications to propagate automatically.
    - **How it Works:** It scans the network for other vulnerable machines, exploits the vulnerability, and copies itself to the new victim. It does not need to attach itself to another program.
    - **Primary Trait:** Self-propagating. It can spread incredibly quickly, causing widespread network congestion and damage.
    - **Example:** The WannaCry ransomware was also a worm. It used the EternalBlue SMB vulnerability to spread automatically to other Windows computers on the same network.

3.  **Trojan Horse (or Trojan):**
    Named after the Greek myth, a Trojan is malicious software disguised as legitimate or desirable software. It tricks users into installing it themselves.
    - **How it Works:** A user is tricked into downloading and running a file they believe is harmless, like a game, a crack, or a utility. Once executed, the Trojan provides a backdoor, installs a keylogger, or downloads other malware.
    - **Primary Trait:** Deception. It does not self-replicate; its spread relies entirely on users being tricked.
    - **Example:** A fake Adobe Flash Player installer that actually installs a backdoor.

4.  **Ransomware:**
    This is a type of malware that encrypts the victim's files, making them inaccessible. The attacker then demands a ransom payment (usually in cryptocurrency) in exchange for the decryption key.
    - **How it Works:** Once executed, it silently encrypts user documents, photos, databases, and other important files. It then displays a ransom note with payment instructions.
    - **Primary Trait:** Denies access to the victim's own data for extortion.
    - **Example:** WannaCry, Locky, Ryuk.

5.  **Spyware:**
    As the name implies, this malware is designed to spy on the user. It gathers information about a person or organization without their knowledge.
    - **How it Works:** It can log keystrokes (keylogger), capture screenshots, monitor browsing habits, and harvest personal information.
    - **Primary Trait:** Covert information gathering.
    - **Example:** A keylogger secretly installed on a point-of-sale system to steal credit card numbers.

6.  **Rootkit:**
    This is a particularly stealthy type of malware designed to gain root-level (administrative) access to a system while hiding its presence and the presence of other malware.
    - **How it Works:** A rootkit modifies the core system components of the operating system (the kernel) or firmware. This allows it to hide files, processes, and network connections from the user and from antivirus software.
    - **Primary Trait:** Advanced stealth and persistence. Removal often requires specialized tools or a complete OS reinstall.
    - **Example:** The Sony BMG copy protection rootkit, which was installed from music CDs to prevent copying but also hid itself and created a security vulnerability.

7.  **Backdoor:**
    A backdoor is any method that bypasses normal authentication to gain remote access to a computer. While it can be a standalone tool, it is often a component of a Trojan or virus.
    - **How it Works:** It creates a network service that allows the attacker to connect and control the machine. Metasploit's Meterpreter payload is a form of a backdoor.
    - **Primary Trait:** Provides persistent, remote, and often hidden access.
    - **Example:** A web shell uploaded to a compromised web server.

8.  **Botnet/Zombie:**
    When a malware infection turns a computer into a "bot" (or zombie), it becomes part of a network of compromised machines (a botnet) controlled by an attacker (the bot-herder).
    - **How it Works:** The bot connects to a C2 server and waits for instructions. The bot-herder can then command all the bots to simultaneously launch a DDoS attack, send spam, or mine cryptocurrency.
    - **Primary Trait:** Turns the victim's machine into a soldier in a large, remote-controlled army.

Understanding these categories allows you to anticipate an attacker's goals and choose the appropriate defensive measures.

Glossary for Topic 43:

Virus: Malicious code that attaches itself to a host file and requires user action to spread.

Worm: A self-replicating, standalone program that spreads automatically across a network by exploiting vulnerabilities.

Trojan Horse: Malicious software disguised as legitimate software, relying on user deception to be installed.

Ransomware: Malware that encrypts a victim's data and demands a ransom for decryption.

Spyware: Malware designed to covertly gather information about a user or organization.

Rootkit: A stealthy type of malware that modifies system components to hide its existence and other malicious activity.

Backdoor: A method, often a hidden network service, that allows an attacker to bypass normal authentication and maintain remote access.

Botnet: A network of private computers infected with malicious software and controlled as a group without the owners' knowledge, e.g., to send spam or launch DDoS attacks.
